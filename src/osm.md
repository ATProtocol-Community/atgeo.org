# OpenStreetMap

Collection: `org.atgeo.places.osm`

ATGeo imports points of interest from [OpenStreetMap](https://www.openstreetmap.org/) (OSM), a collaborative open-source mapping project. OSM uses a flexible tagging system described in its [data model](https://wiki.openstreetmap.org/wiki/Elements) and [map features](https://wiki.openstreetmap.org/wiki/Map_features) documentation.

OpenStreetMap support is under development.
